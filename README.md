## Prerequisites

- node >=16.0.0
- yarn >=3.0.0

## Install

```sh
yarn install
```

## Usage

### Development

```sh
yarn dev
```

or

```sh
yarn dev:all
```

### Create a new plugin

```sh
yarn create-plugin
```

### Build

```sh
yarn build
```

or

```sh
yarn build:all
```

### Check packages

```sh
yarn check-packages
```
